# Frosty - Your Penguin AI Companion

## Setup Instructions

1. Create a `public` directory in your project root:

```bash
mkdir public
```

2. Add your intro video:

   - Create or obtain an MP4 video file for Frosty's introduction
   - Name it `intro.mp4`
   - Place it in the `public` directory
   - Recommended video specifications:
     - Duration: 3-5 seconds
     - Resolution: 1280x720 or 1920x1080
     - Format: MP4
     - Content suggestion: A cute penguin animation with a welcoming message

3. The intro video will automatically play when users first visit the website. After the first visit, it will be skipped unless the user clears their browser data.

## Features

- Friendly penguin AI companion
- Voice interaction
- Joke telling
- Dark/Light mode
- Customizable voice settings
- Responsive design
